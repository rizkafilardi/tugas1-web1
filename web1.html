Rangkuman HTML Basic Tag

Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membangun dan menampilkan halaman web di browser. HTML telah menjadi standar utama dalam pengembangan web yang diatur oleh World Wide Web Consortium (W3C).

Sejarah HTML

  • HTML 2.0 disetujui pada 22 September 1995
  • HTML 3.2 diluncurkan pada 14 Januari 1996
  • HTML 4.0 dirilis pada 18 Desember 1997
  • XHTML 1.0, penggabungan HTML dan XML, diumumkan pada 24 Januari 2000

Ciri-Ciri Dokumen HTML

  • Memiliki ekstensi file berupa .htm atau .html.
  • Bersifat non case-sensitive, artinya tag dapat ditulis dalam huruf besar maupun kecil.
  • Terdiri atas tag pembuka dan penutup, walaupun beberapa tag bisa berdiri sendiri tanpa penutup (seperti <br />).
  • Tag HTML bersifat saling berpasangan dan bersarang, seperti tag <html> yang membungkus seluruh isi dokumen.

Struktur Standar Dokumen HTML

<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        ...konten halaman...
    </body>
</html>

Tag Penting dalam HTML

  • Tag Heading: <h1> sampai <h6> digunakan untuk membuat judul dan sub-judul.
  • Tag Paragraf: <p> digunakan untuk menuliskan paragraf.
  • Tag Gambar: <img src="..." alt="..."> digunakan untuk menambahkan gambar di halaman web.
  • Tag Link: <a href="..."> digunakan untuk membuat hyperlink ke halaman atau file lain.
  • Tag Komentar: <!-- komentar --> digunakan untuk menulis komentar dalam kode HTML.

Tips & Trik Penggunaan Tag HTML

Tips untuk Tag Gambar (<img>)

Pastikan Anda selalu menggunakan atribut alt untuk memberikan deskripsi gambar. Ini membantu dalam SEO dan aksesibilitas.

<img src="logo.png" alt="Logo Perusahaan">
